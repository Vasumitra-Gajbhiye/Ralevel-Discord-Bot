# r/alevel Discord Bot

Advanced moderation and utility bot powering the r/alevel Discord server.

The bot is designed to automate moderation, improve academic collaboration, and provide quick access to learning resources.

## Features

### Subject Helper System

The `/helper` command allows students to request help in specific subject channels.

When used in a subject channel, the bot automatically:
- Identifies the relevant subject
- Pings verified helpers for that subject
- Notifies them that assistance is needed

This enables faster support for students asking academic questions.

### Past Paper Access

The `/pastpaper` command allows users to retrieve past exam papers directly within Discord.

Students can quickly access:
- Subject-specific past papers
- Organized exam resources

### Moderation System

The bot includes a comprehensive set of moderation tools.

Supported features include:
- Kick / ban commands
- Timeout management
- Message moderation
- Role management
- Channel control
- Utility commands

These tools help moderators maintain a healthy and organized community environment.

### Community Automation

Additional automation features support the large-scale r/alevel student community.

The bot is built to handle thousands of users efficiently.

## Tech Stack

- JavaScript
- Node.js
- Discord.js

## Purpose

The goal of this project is to improve collaboration in a large academic Discord community by providing automation tools and resource access directly within the server.
